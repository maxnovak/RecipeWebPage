Desgin Considerations and Aesthetics:
This design was made to focus around the recipes that the cooks are looking for, hence making that the front page. The cooks immediately have the ability to search for a specific recipe. If the cook cannot find a specific recipe they also have the ability to add to the database.
Information exchange with the server:
Ajax is used to call a php script which queries the database for recipes. To add recipes to the database a php script is run to append the data to the database.
